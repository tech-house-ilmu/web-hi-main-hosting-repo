Information :

  • Deskripsi Program : Kesempatan punya pengalaman internasional dari program Pionirmuda Cross-Academy Exchange (PCAE) Gratis ke Singapore dan Malaysia. Program student exchange untuk kamu yang ingin : Naikin value pribadi > Bangun Personal Branding > Punya portofolio > Persiapan lanjut studi > Dapat Endorse > dibimbing dapat sponsor kegiatan.
  • Lokasi Tujuan : Singapore dan Malaysia
  • Durasi program : 3-6 Agustus 2025
  • Jenis Program : Exchange
  • Biaya Hidup/beasiswa : Uang Saku Program, Konsumsi, dan Transportasi

    3. Dokumentasi Program dan Delegasi
